# LCPlus 学校系统界面美化

## 简介

LCPlus 是一个纯 CSS 美化工具，让学校课程系统（LinkedClassroom）界面更清晰、更舒适。

- ✅ 纯 CSS，无脚本
- ✅ 不影响任何功能
- ✅ 不收集任何数据
- ✅ 刷新页面即可恢复

## 使用方法

**第一步：安装 Stylus 插件**

- Chrome：[安装](https://chrome.google.com/webstore/)
- Edge：[安装](https://microsoftedge.microsoft.com/addons)

**第二步：复制主题代码**

点击上方任意「复制代码」，全选复制全部内容

**第三步：粘贴生效**

1. 点击浏览器右上角 Stylus 图标
2. 选择「管理」→「编写新样式」
3. 粘贴代码
4. 设置应用到 `linkedclassroom.com`
5. 保存

刷新学校页面即可看到效果。

## 更换主题

在 Stylus 中删除当前样式，按上述步骤安装新主题即可。

## 卸载

在 Stylus 管理中删除该样式，或直接禁用 Stylus 插件。
